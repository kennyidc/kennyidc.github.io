# 英语学习效果评估系统

## 系统功能

本系统是一个离线的英语学习效果评估网页应用，专为文科大学生完成英语学习效果评估课题而设计。系统支持以下功能：

1. **用户登录及管理模块**：
   - 管理员登录：使用管理员账号登录后可录入所有用户账号密码
   - 用户管理：录入用户时可添加备注字段，方便对应用户名与学生的真实姓名
   - 学生登录：使用管理员录入的账号密码登录系统

2. **习题库模块**：
   - 管理员可录入或管理习题，习题按期数存放
   - 每个习题必须设置时限，控制答题时间
   - 支持单选题（二选一）、多选题（四选1-4）、拼写填空题
   - 多选题可添加音频字段，测验时进入该题目后自动播放音频
   - 答题时限需要大于音频时长（如果是多选题且有音频）

3. **习题测试模块**：
   - 答题引导页面：每一类型的题目开始前显示答题方式和注意事项
   - 学生登录后选择测试期数进行答题
   - 所有习题都有时限要求，时限结束后立即翻页进入下一习题，无论是否答题
   - 有时限要求的习题会显示在限时页面，显示倒计时
   - 多选题进入后会自动播放音频
   - 支持上一题、下一题导航

4. **答卷输出模块**：每位学生测验结束后，系统自动发送学生的答案到指定邮箱，若发送失败，则提醒学生检查网络并点击"重新上传"按钮，系统再次发送邮件

## 技术栈

- **前端**：HTML5, CSS3, JavaScript
- **Excel处理**：SheetJS库（通过CDN引入）
- **数据存储**：浏览器本地存储（localStorage）

## 使用步骤

### 1. 管理员操作

#### 1.1 管理员登录
1. 双击`index.html`文件在浏览器中打开系统
2. 在登录页面输入管理员账号：
   - 用户名：`silvia`
   - 密码：`527070`
3. 点击"登录"按钮进入系统

#### 1.2 用户管理
1. 登录后，点击"用户管理"按钮进入用户管理界面
2. 在"添加用户"表单中填写：
   - 用户名：学生的登录账号
   - 密码：学生的登录密码
   - 备注（真实姓名）：学生的真实姓名
3. 点击"添加用户"按钮保存用户信息
4. 在"用户列表"中查看已添加的所有用户
5. 点击"返回测试"按钮回到主界面

#### 1.3 习题管理
1. 登录后，点击"习题管理"按钮进入习题管理界面
2. 在"添加习题"表单中填写：
   - 期数：习题所属的测试期数
   - 问题内容：习题的具体内容
   - 题型：选择单选题、多选题或填空题
   - 选项A-D：根据题型填写相应的选项
   - 音频路径：多选题的音频文件路径（可选）
   - 时限（秒）：答题时间限制，必须大于0秒
3. 点击"添加习题"按钮保存习题信息
4. 在"习题列表"中选择期数查看已添加的习题
5. 点击"返回测试"按钮回到主界面

#### 1.4 邮箱配置
1. 登录后，点击"邮箱配置"按钮进入邮箱配置界面
2. 在"邮箱设置"表单中填写：
   - SMTP服务器：邮箱的SMTP服务器地址（如smtp.qq.com）
   - SMTP端口：邮箱的SMTP端口（如587）
   - 邮箱账号：用于发送邮件的邮箱账号
   - 邮箱密码/授权码：用于发送邮件的邮箱密码或授权码
   - 接收邮箱：接收测试结果的邮箱地址（默认：1585088657@QQ.COM）
3. 点击"保存配置"按钮保存邮箱配置
4. 点击"返回测试"按钮回到主界面

### 2. 学生操作

#### 2.1 学生登录
1. 双击`index.html`文件在浏览器中打开系统
2. 在登录页面输入管理员分配的用户名和密码
3. 点击"登录"按钮进入系统

#### 2.2 进行测试
1. 在测试页面的下拉菜单中选择测试期数
2. 点击"开始测试"按钮进入答题界面
3. 每一类型的题目开始前会显示答题引导页面，阅读答题方式和注意事项后点击"我知道了"按钮开始做题
4. 按顺序完成所有题目：
   - 单选题（二选一）：点击选择正确答案
   - 多选题（四选1-4）：点击选择一个或多个正确答案，进入题目后会自动播放音频
   - 拼写填空题：在空白处输入正确的单词
   - 所有习题都有时限要求，在规定时间内完成答题，时间到后自动进入下一题目，无论是否答题
5. 可使用"上一题"和"下一题"按钮导航
6. 完成后点击"提交试卷"按钮查看结果

#### 2.3 查看结果
1. 在结果页面查看答题情况
2. 系统会自动发送测试结果到指定邮箱
3. 若发送失败，会提醒您检查网络并点击"重新上传"按钮，系统会再次发送邮件
4. 点击"导出答卷"按钮可将结果导出为Excel文件（可选）
5. 导出的文件命名格式为：`用户名_测试期数_答卷.xlsx`

### 3. 退出系统

点击"退出登录"按钮返回登录页面

## 注意事项

1. 系统为离线应用，无需网络连接即可使用
2. 音频文件需与HTML文件放在同一目录或指定正确的相对路径
3. 建议使用现代浏览器（如Chrome、Firefox、Edge）以获得最佳体验
4. 导出的答卷文件可直接用于后续的学习效果分析
5. 用户数据和习题数据存储在浏览器本地，清除浏览器数据会导致信息丢失
6. 多选题进入后会自动播放音频，请确保设备音量适中
